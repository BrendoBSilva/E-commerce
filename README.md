📊 Dashboard de E-commerce | Análise de Dados e Marketing
🚀 Visão Geral

Este projeto tem como objetivo analisar a performance de um e-commerce utilizando dados de vendas, clientes, produtos e campanhas de marketing.

A proposta é simular o trabalho de um analista de dados, respondendo perguntas como:

Como está o faturamento do negócio?
Quais produtos geram mais receita?
O investimento em marketing está trazendo retorno?
Quem são os clientes mais valiosos?
🧱 Estrutura dos Dados

O projeto utiliza quatro principais fontes de dados:

Orders (Pedidos): informações de vendas (data, valor, quantidade)
Customers (Clientes): dados dos clientes
Products (Produtos): categorias e preços
Ads (Marketing): dados de campanhas (custo, cliques, conversões)
🛠️ Ferramentas Utilizadas
SQL → consultas e transformação de dados
Python (pandas) → limpeza e análise
Microsoft Power BI → visualização e dashboard
📊 Dashboard

O dashboard foi estruturado em quatro páginas principais:

📌 1. Visão Geral

Apresenta os principais KPIs do negócio:

Receita Total
Total de Pedidos
Clientes
Ticket Médio
CAC (Custo de Aquisição)
ROI (Retorno sobre investimento)

Inclui também:

evolução da receita ao longo do tempo
filtros por data, cliente e canal
🛒 2. Produtos

Foco na performance de vendas:

produtos mais vendidos
receita por categoria
análise de volume de vendas
📣 2. Marketing

Análise da eficiência dos canais de aquisição:

custo por canal
receita por canal
ROI por canal
comparação entre investimento e retorno

🧠 Insights Obtidos
A receita apresentou crescimento ao longo do período analisado
Alguns canais apresentam alto custo e menor retorno, indicando baixa eficiência
Clientes recorrentes possuem maior contribuição no faturamento
Existe concentração de receita em determinados produtos
